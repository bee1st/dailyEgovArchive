# dailyEgovArchive-web

서비스(사용자) 화면 모듈.

## Tech Stack
- eGovFramework + Spring MVC
- JSP / JSTL
- MyBatis
- PostgreSQL
- Apache Tiles

## Local Setup (WIP)
- DB: `egovArchive_db`
- User: `egovArchive`
- URL (예): `/notice/list.do`
